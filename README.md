# 健身房管理系统 java164

## 项目概述

健身房管理系统是一款基于Java和SpringBoot框架，采用前后端分离架构的信息管理系统。系统前端使用Vue技术，后端采用Java语言，搭配Mybatis持久层框架和Mysql数据库。

## 技术栈

### 系统架构
- B/S架构
- MVC模式

### 系统环境
- Windows/Mac

### 开发环境
- IDEA
- JDK 1.8
- Maven
- Mysql

### 技术组件
- Java
- Mysql 8.0
- SpringBoot
- Mybatis
- Vue

## 功能模块

### 用户角色

#### 管理员
- 会员管理
- 健身房管理
  - 课程管理
  - 器材管理
- 员工管理
- 报名表管理

#### 会员
- 课程查询
- 在线充值
- 充值记录
- Vip管理
- 购买记录
- 我的资料
- 修改密码

## 运行环境

- 服务器：符合Java应用运行标准的服务器
- 数据库：Mysql 8.0及以上版本
- 客户端：现代浏览器

- 前后端分离，易于维护和扩展
- 基于SpringBoot，开发效率高

## 部署步骤

1. 环境准备：确保服务器满足运行要求，安装JDK 1.8、Mysql数据库、Maven等
2. 数据库部署：执行数据库脚本，创建数据库和表结构
3. 后端部署：编译后端代码，部署到Java应用服务器
4. 前端部署：编译前端代码，部署到Web服务器
5. 配置网络访问：确保前端可以访问后端API
6. 运行系统：启动前后端应用，进行系统测试
